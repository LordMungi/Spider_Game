# Web Cleaner

## Synopsis
An arcade-style game about destroying enemies using a pendular motion to stop them from cutting your string.
- _Pendulum is affected by gravity_
- _Ability to "push" the pendulum_
- _Variable string length_
- _Enemies will spawn consistently_
- _Destroy enemies by hitting them with the ball_
- _If an enemy touches the string, you die_

## Controls
Key|Action
-|-
W/Up|Shorten string
S/Down|Lengthen string
A/Left|Push left
D/Right|Push right
ESC|Pause

Uses SFML and C++
